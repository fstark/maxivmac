This is maxivmac, a modernisation of the minivmac mac emulator

It supports emulating vintage macs, form the 128k to the MacII across a variety of platforms.

See [docs/GLOSSARY.md](docs/GLOSSARY.md) for project terminology.

In reference/ there is a branch that contains the original project with some changes suitable for non-regression testing.

NEVER USE `git add -A` OR `git add .` — ONLY STAGE THE SPECIFIC FILES YOU CHANGED.
